# Sneha-chandra Portfolio

This repository contains the source code for my personal portfolio website. The site showcases select projects, skills, and my professional story, with a focus on software development, frontend/backend technologies, and project documentation.

## Features

- **Responsive Design**: Mobile-friendly layout and accessibility enhancements.
- **Dynamic Project Listings**: Displays GitHub projects with descriptions and repo links.
- **Skills & Tech Stack**: Highlights experience in Python, JavaScript, HTML/CSS, and frameworks.
- **Contact Section**: Easy-to-use contact form for outreach and networking.
- **Deployed on GitHub Pages**: Live, shareable site for recruiters and collaborators.
